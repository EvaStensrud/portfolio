an alphabet

This is the final project I made for the CAS Coding for the Arts I followed in 2021/2022 at the ZHdK. In 26 sketches, following the lettres of the latin alphabet, it allows an insight into what I learned during the programme. Languages and concepts I used are Drawbot / Python, javaScript, p5js, SVG, Simplex Noise, Perlin Noise, webGL, Machine Learning with xxx and HTML / CSS. Want to learn more about me?
